\chapter{Conclusiones}
%Con este trabajo se pudo observar que el mundo de las redes neuronales y sobre todo las profundas, es un mundo totalmente infinito en el cual existen infinidad de combinaciones entre los hiperparámetros que estas poseén, sin embargo y a pesar de esta adversidad, 
Con este trabajo se obtuvo un modelo tanto para la esfericidad como para la redondez el cual es invariante a la escala, rotación, y traslación utilizando la curvatura y el análisis de frecuencias en una red neuronal profunda, además de ser mucho más rápido para calcular la redondez.

La red neuronal de la esfericidad por si sola obtuvo un muy buen resultado clasificando de manera muy óptima el grado de las partículas, por ser un término muy simple. Para el caso de la redondez para el caso de 5 clases, se logró obtener una red la cual arroja que el 70\% de las partículas de las pruebas estuvo debajo del \(\pm 0.1\), algo que nos interesa porque es la varianza entre las clases y que se puede predecir el 70\% de las veces de manera correcta, además de que está muy cerca al valor que los geológos obtienen al clasificar rocas sedimentarias, que es el 80\%, siendo el nuestro un valor muy aceptable en el campo geológico, unos resultados bastante buenos. A medida que se va aumentando la cantidad de clases, el error cada vez va a ser mayor.
